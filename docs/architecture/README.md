# Mimari Belgeler

Bu klasör, sistem bileşenleri ve aralarındaki kalıcı ilişkiler için yönlendirme noktasıdır.

Şu anda ayrıntılı mimari, veri modeli, API listesi, deployment ve GMİM/DUİM yapısı kökteki [`SISTEM_DOKUMANTASYONU.md`](../../SISTEM_DOKUMANTASYONU.md) dosyasında tutulmaktadır. Bu legacy belge aşamalı olarak daha küçük ve tek sorumluluklu mimari/runbook belgelerine ayrılacaktır.

Güncel tamamlanma durumu için teknik doküman veya eski planlar değil, [`docs/status/CURRENT.md`](../status/CURRENT.md) kullanılır.
